This is the platform's design system, rendered from its own tokens. Every swatch below is a live CSS variable, every specimen is set in the shipping typeface at the shipping size, and the component gallery renders the real shared components — not screenshots. Toggle between light and dark to watch the token system flip.

Three principles hold it together: a warm neutral palette with no pure grays, a muted feedback set that shows outcomes rather than verdicts, and one dedicated action color kept deliberately outside the 4D competency palette — so the competency colors never lose their meaning.

## A note on the values

The swatches show the values as implemented, which is the source of truth. A few low-emphasis neutrals — `tertiary`, `muted`, `subtle`, and `ghost` — were darkened from the original specification during a WCAG 2.1 AA pass so they clear a 4.5:1 contrast ratio against the warm canvas; those are flagged on their swatches. Everything else matches the design-system document exactly, including the exact type scale and the three-typeface stack (DM Serif Display, DM Sans, DM Mono).
